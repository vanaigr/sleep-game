### Переход в настройки
Переход в настройки осуществляется с помощью кнопки в правом верхнем углу экрана.
При нажатии на кнопку, открывается новый экран, где расположены все настройки.

### Базовые шаблоны
#### Поле для изменения значения
Состоит из текста, содержащего название переменной, и кнопки, отображающей текущее значение переменной.
![[Pasted image 20260608100926.png|363]]
При нажатии на кнопку всплывает окно, где пользователю предлагается изменить значение переменной.
![[Pasted image 20260608101311.png|406]]
При нажатии на кнопку "ок" - значение сохраняется и окно закрывается.
При нажатии на кнопку "отмена" - значение не меняется и окно закрывается. 

#### Кнопка действия
[[#Навигация по настройкам]]
Состоит из названия действия и кнопки "Выполнить".
![[Pasted image 20260608105615.png|355]]
При нажатии на кнопку "Выполнить" всплывает окно, где пользователю предлагается подтвердить выполнение действия.
![[Pasted image 20260608105857.png|338]]
При нажатии на кнопку "Да" действие выполняется и окно закрывается.
При нажатии на кнопку "Нет" действие не выполняется и окно закрывается.

#### Поле включить/выключить
[[#Навигация по настройкам]]
Состоит из названия того, что нужно включить/выключить, и кнопки-флага.
![[Pasted image 20260608111757.png|334]]
При нажатии на кнопку значение флага меняется на противоположное. Картинка галочки / крестика на кнопке меняется на противоположную.

### Быстрые кнопки 
[[#Навигация по настройкам]]
[[Экран блокировки#Быстрые кнопки для записи сна|О быстрых кнопках]]

- Быстрые кнопки в уведомлении на заблокированном экране — тип [[#Поле включить/выключить]]. По умолчанию: включено.
- Звук нажатия на быстрые кнопки — тип [[#Поле включить/выключить]]. По умолчанию: включено.
- Вибрация нажатия на быстрые кнопки — тип [[#Поле включить/выключить]]. По умолчанию: включено.

- Порядок быстрых кнопок (1 - подъём / 2 - ночное пробуждение / 3 - отбой) — тип [[#Поле для изменения значения]]. По умолчанию: 123
- Размер кнопки "подъём" — тип [[#Поле для изменения значения]]. По умолчанию: 20%
- Размер кнопки "ночное пробуждение" — тип [[#Поле для изменения значения]]. По умолчанию: 60%
- Размер кнопки "отбой" — тип [[#Поле для изменения значения]]. По умолчанию: 20%
